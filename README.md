# DevPocket

Bootstrap a professional GitHub repository in seconds.

DevPocket is a simple CLI tool that helps developers quickly initialize a new project with essential files like:

- README.md
- LICENSE
- .gitignore

## Installation

### Global install

```bash
npm install -g devpocket
```
### Using npx
```bash
npx devpocket init
```
### Usage
Initialize a project in the current directory:
```bash
devpocket init
```
Initialize in a specific directory:
```bash
devpocket init --dir my-project
```
### Example
```bash
devpocket init
? Project name: my-app
? Project description: My first project
? Choose license: MIT
```
